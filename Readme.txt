~~~~~~~~~~~~~~~~~~~~~~~~~~~~         Nelson        ~~~~~~~~~~~~~~~~~~~~~~~~~~~~
~~~~~~~~~~~~~~~~~~~~~~~~~~~~ The Virtual Assistant ~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Nelson is a virtual assistant that respond to the user in regional languages.
As of now Nelson can speak in Tamil and English.

The default voice is English.
By changing the voice to Tamil lets Nelson speak in Tamil.

Add voices to Nelson:
    > Open the Voices folder.
    > Then Double click (or) right click and select 'Merge'.
    > Now Nelson can speak in that language.
